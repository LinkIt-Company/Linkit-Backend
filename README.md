## Generate prisma client (TS types)

```
pnpm pg:generate
```

## Project Init

- Local Postgres Up/Down

```
pnpm (`pg:up` | `pg:down`)
```

- 마이그레이션 스크립트 생성

```
pnpm pg:makemigrations (마이그레이션 이름)
```

- 마이그레이션 sync

```
pnpm pg:migrate
```
